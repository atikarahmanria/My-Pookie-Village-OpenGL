# Welcome to My Pookie Village 🏡

A visually rich and highly interactive 2D Computer Graphics project built using C++ and OpenGL, representing a beautiful, dynamic rural village environment with advanced graphics algorithms, weather effects, and real-time user interaction.

---

## 📖 Project Overview

**My Pookie Village** demonstrates a living, breathing animated village environment. The project seamlessly integrates environmental changes, custom rendering techniques, and interactive controls to simulate realistic rural life with real-time feedback.

---

## ✨ Key Features

* **🌅 Environmental Transitions:** Day, Night, and Rainy modes with dynamic background lighting.
* **🌧️ Real-time Weather Effects:** Particle-based dynamic rain system that overrides both day and night environments.
* **🦅 Swan Wildlife:** Flocks of birds with controllable 2D vector movement.
* **⛵ Real-time Navigation:** Interactive boat system responding directly to mouse inputs.
* **💡 Smart Lighting & Objects:** Automatic moon/stars toggle, glowing house windows, and synchronized lighting effects.
* **🎵 Audio Integration:** Dedicated background audio track with user-controlled mute/unmute toggles.

---

## 🎮 User Controls & Interaction

The application provides a fully interactive experience through both keyboard and mouse inputs, allowing users to dynamically manipulate the environment.

### 🌅 Environment & Sound Controls (Keyboard)
* **D / d** — Switch to **Day Mode** (Resets rain and nighttime).
* **N / n** — Toggle **Night Mode** (Turns on the moon, stars, and house lights).
* **R / r** — Toggle **Rain Mode** (Dynamic particle effect over day/night).
* **M / m** — **Mute** background audio.
* **U / u** — **Unmute** and play environmental sound.

### 🦅 Bird Movement Controls (Arrow Keys)
* **↑ (Up Arrow)** — Move birds upward.
* **↓ (Down Arrow)** — Move birds downward.
* **← (Left Arrow)** - Move birds to the left.
* **→ (Right Arrow)** — Move birds to the right.

### ⛵ Boat Navigation Controls (Mouse Clicks)
* **Left Click** — Move the boat from **Left to Right**.
* **Right Click** — Move the boat from **Right to Left**.
* **Middle Click (Scroll Button)** — **Stop** boat movement immediately.

## 🧮 Algorithms & Graphics Concepts

### 📐 Implemented Algorithms
* **DDA Line Algorithm:** Used for pixel-perfect structural layouts, fences, and straight-line boundaries.
* **Midpoint Circle Algorithm:** Utilized for generating perfectly proportioned sun, moon, and spherical landscape structures.
* **Bezier Curves:** Specifically implemented to render smooth curved layouts including the **village road**, organic **banyan tree trunks**, and natural **coconut tree shapes**.

### 🧠 Core Concepts Covered
* **2D Graphics Rendering:** Matrix operations and primitives configuration.
* **Transformations:** Implementation of **Translation, Rotation, and Scaling** for object states.
* **Real-time Animation:** Frame-by-frame updates for environmental loops.
* **Event-driven Programming:** Seamless handling of keyboard, special arrow keys, and mouse button events.
* **Alpha Blending:** Used for smooth rain opacity, night shadows, and lighting transparencies.
